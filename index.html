Global Data
Global Data is a 3d data visualization of international trade.
Built using Vanilla Javascript, Three.js, and Geometry.
Instructions:
  • Use the country searchbar to change countries
  • Use the product searchbar to change products
  • Use the year range at the bottom to change the range
  • Thicker arcs incdicate more trade
  • Blue arcs represent imports and red arcs represent exports
  • Press any key or click to begin
